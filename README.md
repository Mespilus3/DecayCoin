# DecayCoin (DKCN)

DecayCoin is a fixed-supply ERC20-style token deployed on Polygon with built-in **2% decay** on every transfer:
- 98% of the amount goes to the recipient
- 2% is permanently burned (reducing total supply)

## Token Details
- **Name**: DecayCoin
- **Symbol**: DKCN
- **Decimals**: 18
- **Total Supply**: 1,100,000,000 DKCN (minted to deployer)

## Features
- Transfer with 2% burn (deflationary)
- Burn your own tokens (`burn`)
- Burn tokens from others with allowance (`burnFrom`)
